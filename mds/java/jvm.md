# jvm

## 整数在Java虚拟机中的比阿诗

原码 -- 符号位加上数字的二进制表示

对于原码来说，绝对值相同的正数和负数只有符号位不同

反码 -- 在原码的基础上，符号位不变，其余位取反

补码 -- 负数的补码就是反码加1，整数的补码就是原码本身


### 使用补码作为计算机内存储方式的好处

1. 可以统一数字0的表示，不管作为正数还是负数，用补码表示都正确
2. 使用补码可以简化整数的加减法计算
